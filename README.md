 # 핵심 기술 문서
 [링크](https://developstudy.tistory.com/72) 클릭 시, 해당 문서로 이동합니다.
  
| 태그         | 의미                            |
| ---------- | ----------------------------- |
| `feat`     | 새로운 기능 추가                     |
| `fix`      | 버그 수정                         |
| `refactor` | 리팩토링 (기능 변화 없음)               |
| `chore`    | 빌드, 설정, 문서 등 기타 작업            |
| `docs`     | 문서 작업                         |
| `style`    | 코드 포맷팅, 세미콜론 누락 등 기능 변화 없는 수정 |
| `test`     | 테스트 추가/수정                     |
| `perf`     | 성능 개선                         |
| `ci`       | CI 관련 설정 변경                   |
